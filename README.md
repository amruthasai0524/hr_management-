# HR Management System using MySQL

## Project Overview

The HR Management System is a relational database project developed using MySQL Workbench. It simulates how organizations manage employee information, departments, payroll, attendance, recruitment, training, projects, and other HR-related operations.

This project was created to strengthen SQL skills by designing a normalized database with proper relationships and realistic sample data.

---

## Objectives

* Design a relational HR database.
* Create multiple interconnected tables.
* Implement Primary Key and Foreign Key relationships.
* Populate tables with sample HR data.
* Build a database suitable for practicing SQL queries.
